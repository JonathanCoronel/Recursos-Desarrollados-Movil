# Recursos-Desarrollados-Movil
En este repositorio se encuentran alojado el proyecto que se ha venido trabajando en clases, a continuación, se muestran algunas de las imágenes de las pantallas que se desarrollaron.

En el proyecto se desarrollo un Splash y un Menú principal que es la botonera desde donde nos podemos mover a las distintas secciones.
